# Bug Report: AudioManager Missing

## ID
BUG-002

## Severity
🔴 Critical - Crash/data loss

## Environment
- Godot Version: 4.6
- Platform: All
- Commit: current

## Description
The AudioManager autoload script is referenced in `project.godot` but the file does not exist. This will cause the game to crash on startup.

## Reproduction Steps
1. Open project in Godot
2. Run the game
3. Observe crash: "Cannot autoload 'res://scripts/systems/audio_manager.gd'"

## Expected Behavior
Game should start with AudioManager loaded as singleton.

## Actual Behavior
Game crashes on startup due to missing script file.

## File Reference
```
[autoload]
...
AudioManager="*res://scripts/systems/audio_manager.gd"
```

## Required Implementation
- Background music playback
- SFX management
- Volume control
- Audio bus configuration
