# Bug Report: Audio Bus Layout Missing

## ID
BUG-005

## Severity
🟡 Major - Feature broken

## Environment
- Godot Version: 4.6
- Platform: All
- Commit: current

## Description
The default audio bus layout file is referenced in project.godot but does not exist.

## Reproduction Steps
1. Run the game
2. Check Output panel for audio errors
3. Audio may not work correctly

## Expected Behavior
Audio buses are properly configured with master, music, and SFX channels.

## Actual Behavior
Missing bus layout - audio may fail or use default Godot configuration.

## File Reference
```
[audio]
buses/default_bus_layout="res://assets/audio/default_bus_layout.tres"
```

## Required File
Create `assets/audio/default_bus_layout.tres` with:
- Master bus
- Music bus (ducked when SFX plays)
- SFX bus
- Ambient bus (for environmental sounds)
