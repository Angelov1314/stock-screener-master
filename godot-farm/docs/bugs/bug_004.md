# Bug Report: Crop Sprites Missing

## ID
BUG-004

## Severity
🟡 Major - Feature broken

## Environment
- Godot Version: 4.6
- Platform: All
- Commit: current

## Description
All 20 crop sprite files defined in asset_manifest.json are missing. Crops will be invisible in-game.

## Reproduction Steps
1. Plant any crop
2. Observe invisible/no crop visual
3. Check Output panel for texture load errors

## Expected Behavior
Crops display with proper sprites for each growth stage.

## Actual Behavior
Crops are invisible (fallback code creates colored squares in FarmController, but sprites fail to load).

## Workaround
FarmController._update_crop_sprite() creates colored placeholder squares, so crops won't be completely invisible.
