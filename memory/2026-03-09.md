## Nano Banana 动物精灵图生成 Prompt

**模板** (替换 `__` 为动物名称，如 cow, sheep, pig, chicken 等):

```
Create a clean 2D game sprite sheet of a cute cartoon __ （cow） in a soft cozy farm style.

Requirements:
- green background (easy for background removal)
- no UI panel
- no wooden frame
- no labels
- no text
- no decorative background
- no shadows outside the character except subtle grounded shadow if needed
- production-ready game asset
- consistent character design across all frames
- same proportions, same face, same body markings in every frame
- each animation on a separate horizontal row
- evenly spaced frames
- grid-aligned layout
- each frame centered in its own cell
- same frame size for all cells
- clean empty padding between rows and columns
- sprite sheet only

Animation rows:
Row 1: idle, 4 frames
Row 2: walking side view, 4 frames
Row 3: happy reaction, 4 frames
Row 4: sleeping, 2 frames
Row 5: carried / picked up pose, 2 frames

Important:
- keep all frames fully visible
- do not merge frames together
- do not crop body parts
- do not vary camera angle
- do not add props
- side view for movement animations
- front or 3/4 front view only for idle or reaction if needed, but stay consistent
- output as a clean sprite sheet ready for slicing
```

**已生成的动物**: cow ✅, sheep ✅
